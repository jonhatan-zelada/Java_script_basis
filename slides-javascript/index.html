Programación modular

La programación modular es una metodología de programación que permite construir un programa grande descomponiéndolo en pequeños subprogramas o módulos.

Variables locales y globales

Los métodos o funciones pueden utilizar sus propias variables denominadas variables locales o variables de uso compartido, comunes a todos los métodos, denominadas variables globales.

Variables locales

Una variable local es una variable que se declara en el interior de un método o funcion que sólo puede ser utilizada dentro del método donde fue declarada. Este tipo de variable se crea al iniciar la ejecución del método y se destruye al finalizar.

Funciones o métodos tipo void

no retorna ningún resultado al punto donde es invocado, razón por el que se le conoce también como método sin valor de retorno

Los métodos tipo void pueden dividirse a su vez en dos tipos:

  • Métodos tipo void sin parámetros.
  • Métodos tipo void con parámetros.

Métodos tipo void sin parámetros.

Este tipo de método se define de la siguiente manera:


                         function nombre(){
                         Declaración de variables locales
                         Cuerpo del método
                     }

Este tipo de método se invoca de la siguiente manera:


                    nombre();

Métodos tipo void con parámetros.

Este tipo de método se define de la siguiente manera:


                 function nombre(p1, p2, p3, ...){
                 Declaración de variables locales
                 Cuerpo del método
             }

Donde:

  • nombre: es el nombre del método
  • p1,p2, p3, ... : Son los nombres de los parámetros

Este tipo de método se invoca de la siguiente manera:


            nombre(v1, v2, v3, ...);

Donde:

  • nombre: es el nombre del método invocado
  • v1,v2, v3, ... : Son los valores de los parámetros

Funciones o métodos con valor de retorno

puede recibir datos de entrada a través de variables locales denominadas parámetros y que retorna un resultado al punto donde es invocado.

Los métodos tipo void pueden dividirse a su vez en dos tipos:

  • Métodos con retorno sin parámetros.
  • Métodos con retorno con parámetros.

Métodos con retorno sin parámetros.

Este tipo de método se define de la siguiente manera:


         function nombre(){
         Declaración de variables locales
         Cuerpo del método
         return valor;
     }

Este tipo de método se invoca de la siguiente manera:


    variable = nombre();

Métodos con retorno con parámetros.

Este tipo de método se define de la siguiente manera:


     function nombre(p1, p2, p3, ...){
     Declaración de variables locales
     Cuerpo del método
     return valor;
 }

Donde:

  • nombre: es el nombre del método
  • p1,p2, p3, ... : Son los nombres de los parámetros

Este tipo de método se invoca de la siguiente manera:


        variable = nombre(v1, v2, v3, ...);

Donde:

  • nombre: es el nombre del método invocado
  • v1,v2, v3, ... : Son los valores de los parámetros

        gato.["nombre"];

Cambiar las propiedades de un objeto

Para cambiar el valor de la propiedad de un objeto, tenemos que indicar el nombre del objeto seguido de un punto, el nombre de la variable agregar el operador de asignación (=) y el nuevo valor de la propiedad 


        gato.nombre="michifus";

otra forma sería usando corchetes


        gato["nombre"]="michifus";
